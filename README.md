# homebrew-tap

Homebrew tap for [Tingly-Box](https://github.com/tingly-dev/tingly-box) - Local AI intelligence layer.

## Installation

```bash
# Tap the repository
brew tap tingly-dev/homebrew-tap

# Install tingly-box
brew install tingly-box

# Run
tingly-box start
```

## Update

```bash
brew upgrade tingly-box
```

## Uninstall

```bash
brew uninstall tingly-box
brew untap tingly-dev/homebrew-tap
```
